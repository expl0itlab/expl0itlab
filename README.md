## Who We Are

**Exploit Lab** is a cybersecurity community focused on **ethical hacking**, **security research**, and **real-world vulnerability discovery**.

We study modern attack surfaces to help defenders build stronger systems.

---

## What We Do

- Web & API security research  
- Broken Access Control (BAC / BOLA / IDOR) analysis  
- Information disclosure & logic flaw discovery  
- Reconnaissance tooling & automation  
- Responsible disclosure & education
